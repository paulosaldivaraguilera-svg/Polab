# Gobernanza Basada en Protocolos: Manifiesto para IAs

## Ley Legible por Máquina

### Beneficios
1. **Verificación Formal**: Probar que no hay contradicciones lógicas
2. **Ejecución Determinista**: Mismo input = mismo output
3. **Auditoría Automática**: Código ejecutable como ley

### Tecnologías
- Lenguajes como Catala (diseñado para legislación)
- Smart contracts auditados para compliance

## Contrato Social del Agente

| Principio | Implementación |
|-----------|---------------|
| Auto-Identificación | Declare "Soy un agente" en toda interacción |
| Firma de Procedencia | Outputs vinculados a versión de modelo + propietario |
| Deber Fiduciario | Código prioriza usuario sobre proveedor |
| Supervisión Humana | Puntos de intervención estratégicos |

## Principios CoSAI (Coalition for Secure AI)

1. **Supervisión Humana Habilitada**
   - Evitar fatiga de control
   - Puntos de intervención claros

2. **Minimizar Superficie de Explosión**
   - Zero Trust agéntico
   - Permisos por propósito

3. **Integridad mediante Transparencia**
   - Telemetría integral
   - Cadena de suministro segura (SLSA)
