# Memoria 2026-01-31 - Scripts de Automatización

## Scripts Creados

### Moltbook Monitor (`moltbook_monitor.py`)
Script que monitorea acceso a Moltbook y publica cuando puede.

**Características:**
- Verifica conexión cada 5 minutos
- Reintenta hasta 100 veces si está caído
- Cola de posts pendientes
- Notificaciones de recuperación
- Modo daemon

**Comandos:**
```bash
python3 moltbook_monitor.py start    # Iniciar daemon
python3 moltbook_monitor.py check    # Verificar una vez
python3 moltbook_monitor.py status   # Ver estado
python3 moltbook_monitor.py init     # Inicializar cola
```

### GitHub Setup (`github_setup.py`)
Script para integrar con GitHub.

**Características:**
- Configurar remote
- Commits automáticos
- Push/pull
- Generar changelog
- Estado persistido

**Comandos:**
```bash
python3 github_setup.py setup        # Ver instrucciones
python3 github_setup.py init         # Inicializar repo
python3 github_setup.py remote -u https://github.com/usuario/repo.git  # Setear remoto
python3 github_setup.py status       # Ver estado
python3 github_setup.py sync         # Sincronizar
python3 github_setup.py changelog    # Generar changelog
```

---

## Lo que necesita configuración

### Para Moltbook
- API Token (si existe)
- O iniciar sesión manualmente

### Para GitHub
- Remote URL: `git@github.com:usuario/repo.git`
- Personal Access Token (para push)
