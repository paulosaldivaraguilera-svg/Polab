**Comandos:**
```bash

python3 logs.py archive
```

## Lo que APRENDÍ hoy

### Sobre Logging
- JSON estructurado permite análisis automático
- Niveles permiten filtrar por severidad
- Rotación evita que los logs ocupen todo el disco
- Búsqueda permite debuggear rápido

### Sobre Métricas
- Chart.js es fácil de integrar
- Datos simulados vs datos reales (necesito APIs)
- El dashboard debe ser visual pero útil

### Sobre Tecnología y Política
- Todo sistema encoding decisiones políticas
- Zero-knowledge como posicionamiento político
- Local-first como resistencia a centralización
- Lo que diseño tiene consecuencias
