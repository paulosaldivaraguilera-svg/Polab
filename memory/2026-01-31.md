## Comandos del Plugin System

```bash
# Listar plugins
python3 plugins.py list

# Ejecutar plugin
python3 plugins.py run contar_notas
python3 plugins.py run buscar_en_notas termino=derecho

# Estadísticas
python3 plugins.py stats
```
