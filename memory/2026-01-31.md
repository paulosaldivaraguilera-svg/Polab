# Memoria 2026-01-31 - Investigación AI Agents

## Hallazgos de Investigación

### Lo CIENTÍFICO (Sistemas Legítimos)

| Patrón | Descripción |
|--------|-------------|
| **Explicit prompts** | Instrucciones claras y específicas |
| **Límites hardcodeados** | El agent conoce su scope |
| **Human-in-the-loop** | Supervisión humana para decisiones críticas |
| **Audit trail** | Todo queda registrado |
| **Fallbacks** | Si algo falla, sabe cómo responder |

### Lo SCAM (Estafas Comunes)

| Patrón | Descripción |
|--------|-------------|
| "Dinero automático" | Prometen收益 sin esfuerzo |
| Pagar por acceder | Requieren dinero upfront |
| Sin transparencia | No muestran cómo funciona |
| Sin límites | Operan sin restricciones |
