# Web Personal - Optimización de Captación

## Análisis de paulosaldivar.cv

### Lo que funciona
- Diseño limpio y profesional
- Zonas claras (Temuco + Online)
- Proceso de trabajo definido
- copy honesto y directo

### Problemas identificados

| Problema | Impacto | Solución |
|----------|---------|----------|
| Solo 3 áreas genéricas | No diferencia propuesta | Segmentar por cliente ideal |
| Sin llamadas a acción claras | Bajo conversion | CTAs agresivos + WhatsApp directo |
| Sin prueba social | Desconfianza | Agregar testimonios, casos |
| Embudo débil | Leads perdidos | Link directo a WhatsApp |
| Solo WhatsApp texto | Fricción | Botón click-to-chat |
| No hay SEO keywords | Visibilidad nula | Optimizar meta tags, H1 |

---

## Plan de Optimización

### 1. Estructura de Landing
```
paulosaldivar.cv/
├── Hero (Valor + CTA)
├── Propuesta Única (¿Por qué yo?)
├── Áreas de Especialidad (3-4)
├── Proceso de Trabajo
├── Prueba Social (Testimonios/Casos)
├── FAQ (Objection handling)
└── CTA Final (WhatsApp)
```

### 2. CTAs a Implementar
- **Botón flotante WhatsApp** en todo el sitio
- **Chat widget** con respuesta automática
- **Formulario simple** nombre + teléfono + tema

### 3. SEO On-Page
- Keywords: "abogado familia Temuco", "herencias Araucanía", "abogado online Chile"
- Meta descriptions optimizadas
- Schema markup para profesionales legales

### 4. Contenido Adicional
- **Guía gratuita** (lead magnet) — "5 cosas que debes saber antes de离婚ar"
- **Blog** con artículos legales básicos
