## Marco General

Este documento establece la infraestructura para gestión de presencia digital en redes sociales, orientada a tres ejes:

1. **Divulgación académica/científica** — Difusión de análisis político, investigación y producción teórica
2. **Desarrollo de negocios** — POLAB, legaltech, captación de clientes
3. **Construcción de identidad intelectual** — Perfil público como abogado marxista, productor teórico, organizador político

## Sistema de Gestión de Contenido

### Flujo de Trabajo

```
1. IDEA/CAPTURA
   → Notas de voz → captura.py
   
2. DESARROLLO
   → writer.py (borrador en produccion/)
   
3. REVISIÓN
   → Aplicar marco de estilo ARIS
   
4. ADAPTACIÓN
   → Versiones para cada plataforma
   
5. PROGRAMACIÓN
   → Buffer/Hootsuite/manual
   
6. PUBLICACIÓN
   → Cross-posting automatizado
   
7. MÉTRICAS
   → Análisis de engagement
```

### Templates por Plataforma

| Plataforma | Formato | Longitud | Elementos |
|------------|---------|----------|-----------|
| Twitter/X | Hilo o tweet único | 280 chars / hilo | Hashtags estratégicos |
| LinkedIn | Artículo largo | 1500-3000 chars | Headers, puntos |
| YouTube | Video | 10-45 min | Descripción, timestamps |
| Instagram | Imagen + caption | 2200 chars | hashtags, CTA |
| Telegram | Texto + medios | Libre | Emojis moderados |

## Notas de Compliance

1. **Transparencia** — Todas las cuentas deben indicar claramente la autoría y afiliación cuando corresponda
2. **Privacidad** — Separación estricta entre vida personal y presencia pública
3. **Consistencia** — Mantener identidad coherenteacross plataformas
4. **Legalidad** — Cumplir términos de servicio de cada plataforma
